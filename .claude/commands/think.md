# חשוב לפני שאתה כותב קוד

לפני כל שינוי, ענה בקול על השאלות הבאות ואז המתן לאישור:

## 1. מה אני מנסה להשיג?
משפט אחד - מה התוצאה הרצויה.

## 2. מה יכול להישבר?
לפחות 3 תרחישים:
- מה קורה אם משהו כבר פתוח / פועל?
- מה קורה אם נתון חסר?
- מה קורה אם ממשק חיצוני השתנה?

## 3. האם יש תלויות?
תוכנה פתוחה, תהליך רץ, הגדרות מחשב ספציפיות, API חיצוני.

## 4. איך מאמתים שזה עבד?
לא שהקוד רץ - אלא שהתוצאה האמיתית נכונה. מה בדיוק בודקים?

## 5. רק אחרי מענה על כל זה - כתוב את הקוד.
